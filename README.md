# 📌 주요 기술

## Singleton

여러가지 필드와 자료들을 중복해서 생성하지 않고 여러 스크립트에서 공유하기위해 정적 객체를 활용.
씬이 전환될 때에도 유지되어야 할 정보를 가진 정적 객체는 DontDestroyOnLoad를 활용하여 객체가 파괴되는 것을 방지.
